Home Network Monitoring Dashboard

Built a Python tool that discovers devices on a network with a ping-sweep, tracks them by MAC address through the ARP table so they stay identified even as DHCP changes their IP, and logs uptime and latency to a Neon PostgreSQL database with threshold-based alerts for outages and slowdowns. Used Nmap to identify device vendors and scan for open ports, then built a Streamlit dashboard with live status, latency charts, and a privacy mode for public sharing.
